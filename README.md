# MINI CRAWLER

## About The Project

Remote controlled vehicle with android app. 
Project  based on NodeMcu v3 dev kit with ESP8266 wifi module on board.

### Built With
* C++ (ArduinoIDE)
* Kotlin (Android Studio)
